API Guide

You can input parameters to the main website to search for different Wikipedia categories and set the number of results to display

For example, the following URL will display the top 5 similarity scores between articles about chocolate bars:
insightfulsolutions.agency?category=Chocolate bars&limit=5

Alternatively, you can use the search button on the main page to filter by category and number of results.
